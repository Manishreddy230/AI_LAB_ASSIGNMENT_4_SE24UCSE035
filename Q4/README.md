## Problem 4: Cryptarithmetic (TWO + TWO = FOUR)

### Description
Each letter represents a unique digit. The goal is to assign digits such that the equation is satisfied.

### Constraints
- Each letter maps to a unique digit
- Leading digits cannot be zero
- TWO + TWO = FOUR must hold true

### Approach
- Generate permutations of digits
- Assign digits to letters
- Check equation validity

### Output
T = 7
W = 3
O = 4
F = 1
U = 6
R = 8

734 + 734 = 1468
